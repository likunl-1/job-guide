# 演示视频目录

## 介绍视频
- **位置**: `intro/`
- **说明**: 项目整体介绍视频（1个）
- **命名规范**: `project-introduction.{mp4|mov|avi}`

## 展示报告视频
- **位置**: `reports/`
- **说明**: 功能展示报告视频（3个）
- **命名规范**:
  - `report-01-{功能名称}.{mp4|mov|avi}`
  - `report-02-{功能名称}.{mp4|mov|avi}`
  - `report-03-{功能名称}.{mp4|mov|avi}`

## 上传说明
1. 将介绍视频放到 `intro/` 目录
2. 将展示报告视频放到 `reports/` 目录
3. 建议视频格式: MP4 (H.264)
4. 建议视频分辨率: 1080p 或 720p
