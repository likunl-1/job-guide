## 功能1：就业市场分析

### 📊 功能描述

帮助用户了解特定行业/职业的就业市场情况，包括趋势、薪资、需求等。

### 🔧 核心能力

#### 1. 行业趋势分析
```
输出内容：
- 行业发展趋势
- 市场需求变化
- 技术演进方向
- 未来发展前景
```

#### 2. 薪资水平分析
```
输出内容：
- 不同经验层级的薪资范围
- 地区薪资差异
- 薪资增长趋势
- 与期望薪资的对比
```

#### 3. 岗位需求分析
```
输出内容：
- 当前招聘需求量
- 主要招聘城市/地区
- 企业类型分布
- 人才缺口情况
```

#### 4. 技能要求分析
```
输出内容：
- 必备技能列表
- 加分技能
- 技能获取难度
- 技能市场价值
```

### 💡 使用示例

#### 示例1：了解行业趋势
```
用户输入：
"前端开发的就业前景怎么样？"

Agent输出：
✅ 前端开发市场需求分析
✅ 2025年趋势预测
✅ 技术栈演进方向
✅ 薪资水平分析
✅ 求职建议
```

#### 示例2：对比不同行业
```
用户输入：
"Java开发和Python开发哪个更有前景？"

Agent输出：
✅ 两个行业的市场需求对比
✅ 薪资水平对比
✅ 技能要求对比
✅ 发展空间对比
✅ 选择建议
```

#### 示例3：地区就业分析
```
用户输入：
"北京和上海的互联网行业发展怎么样？"

Agent输出：
✅ 两地行业规模对比
✅ 薪资水平对比
✅ 生活成本对比
✅ 就业机会对比
✅ 选择建议
```

### 🎯 适用场景

## 功能2：多模式搜索

### 📊 功能描述

提供**浅层**和**深层**两种模式，根据用户需求自动选择最合适的数据获取方式。

### 🔧 两种模式对比

| 特性 | 浅层模式 | 深层模式 |
|-----|---------|---------|
| **数据类型** | 行业报告、趋势分析 | 真实岗位列表、具体职位 |
| **数据来源** | 搜索引擎+AI总结 | 招聘API或本地文件 |
| **响应速度** | 快（几秒） | 慢（需调用API） |
| **数据时效** | 准实时 | 实时/准实时 |
| **费用** | 免费 | 可能收费（API调用） |
| **适用场景** | 了解趋势、薪资概况 | 找工作、投递简历 |

### 💡 浅层模式

#### 适用场景
```
✅ "前端开发的就业前景怎么样？"
✅ "2025年IT行业发展趋势如何？"
✅ "Python开发工程师平均薪资是多少？"
✅ "北京和上海哪个城市就业机会多？"
✅ "什么行业最缺人？"
```

#### 输出示例
```
## 📊 就业市场分析（浅层模式）

**数据来源**：搜索引擎 + AI总结
**适用场景**：了解行业趋势、市场概况

### 🤖 AI智能总结
2025年前端开发市场需求保持强劲，React和Vue框架需求量最大。
一线城市前端工程师平均薪资在15-25K/月，初级岗位起薪10-15K。

### 📋 详细信息
**2025年前端开发就业市场分析报告**
- 来源：拉勾网
- 发布时间：2025-01-15
- 摘要：前端开发岗位需求同比增长20%...
```

### 🎯 深层模式

#### 适用场景
```
✅ "北京有哪些公司在招前端开发？"
✅ "腾讯现在有哪些岗位在招？"
✅ "这个岗位具体薪资多少？"
✅ "我想找一份10K+的前端工作"
✅ "最近一周有哪些新发布的岗位？"
```

#### 输出示例
```
## 📊 就业市场分析（深层模式）

**数据来源**：真实招聘数据
**适用场景**：求职决策、岗位对比

### 📝 找到 10 个相关岗位

**1. 高级前端开发工程师**
- 公司：腾讯科技
- 薪资：25-40K·14薪
- 地点：北京·海淀区
- 经验：3-5年
- 学历：本科及以上
- 职位描述：负责公司核心产品的前端开发...
```

### 🚀 模式选择逻辑

Agent会根据用户提问自动选择模式：

```
用户提问
    ↓
关键词检测
    ↓
包含"趋势"、"前景"、"怎么样" → 浅层模式
包含"招"、"岗位"、"职位" → 深层模式
包含"薪资"、"多少"、"范围" → 浅层模式（概况）/ 深层模式（具体）
    ↓
自动调用相应工具
```

## 功能3：用户画像分析

### 📊 功能描述

根据用户提供的个人信息，生成结构化画像，评估竞争力，识别技能缺口。

### 🔧 分析维度

#### 1. 专业背景分析
```
分析内容：
- 专业与目标岗位的匹配度
- 学历背景竞争力
- 相关课程/证书
```

#### 2. 技能评估
```
分析内容：
- 技能列表与市场需求对比
- 技能熟练度评估
- 技能缺口识别
- 加分技能推荐
```

#### 3. 工作经验分析
```
分析内容：
- 工作年限与目标岗位匹配度
- 相关工作经验权重
- 实习经历价值
- 项目经验含金量
```

#### 4. 兴趣与目标分析
```
分析内容：
- 兴趣方向与市场趋势匹配
- 职业目标合理性
- 发展潜力评估
```

### 💡 使用示例

#### 示例1：应届生分析
```
用户输入：
"我是计算机专业应届生，会Python和Java，想找后端开发工作，在北京"

Agent输出：
✅ 专业匹配度：高（计算机→后端开发）
✅ 技能匹配度：中（Python、Java符合要求）
✅ 工作经验：无（应届生）
✅ 建议岗位：初级后端开发、Java开发工程师
✅ 技能缺口：Spring Boot、MyBatis等框架
✅ 求职策略：从小公司积累经验
```

#### 示例2：转行分析
```
用户输入：
"我是市场营销专业，想转行做数据分析"

Agent输出：
✅ 专业匹配度：低（需要补充技能）
✅ 转行可行性：中（需要系统学习）
✅ 技能缺口：Python、SQL、统计学
✅ 学习路径：3-6个月系统学习
✅ 入门岗位：数据分析师助理
✅ 建议策略：边学边找实习机会
```

### 🎯 适用场景

- ✅ 评估求职竞争力
- ✅ 识别技能缺口
- ✅ 制定学习计划
- ✅ 转行可行性评估
- ✅ 职业规划参考

## 功能6：历史案例引导与心理疏导

### 📊 功能描述

识别用户的消极情绪和对经济的悲观预期，通过历史案例进行心理疏导和引导。

### 🔧 核心能力

#### 1. 消极情绪识别
```
识别信号：
- "经济永远不会好了"
- "我这代人完了"
- "找不到工作，我太失败了"
- "干脆不找工作了"
- "躺平算了"
- "对未来很迷茫"
```

#### 2. 历史案例库

包含丰富的历史案例：

| 案例 | 时间 | 事件 | 困境 |
|-----|------|------|------|
| **美国"消失的一代"** | 2000-2010 | 互联网泡沫+金融危机 | 失业率10%，投200份简历无果 |
| **日本"失去的三十年"** | 1990-2020 | 房地产泡沫破裂 | "冰河期世代"，就业困难 |
| **1929大萧条** | 1929-1939 | 股市崩盘 | 失业率25% |
| **1997亚洲金融危机** | 1997-1998 | 货币贬值 | 企业大量破产 |
| **2020COVID-19** | 2020-2022 | 全球疫情 | 短期失业率飙升 |

#### 3. 引导策略

**六步引导流程**：
```
1. 共情：理解用户情绪
2. 案例：引用历史真实故事
3. 对比：对比当前与历史
4. 规律：说明经济周期
5. 建议：给出应对策略
6. 赋能：给予希望和信心
```

### 💡 使用示例

#### 示例1：极度悲观

```
用户输入：
"我感觉经济太差了，永远找不到工作，我这代人是不是完了？"

✅ 共情理解
✅ 引用美国"消失的一代"案例
✅ 讲述John的真实故事
✅ 说明经济周期规律
✅ 给出应对建议
✅ 强调时间站在你这边
```

#### 示例2：历史对比
```
用户输入：
"听说日本有'失去的三十年'，我们会重蹈覆辙吗？"

Agent输出：
✅ 详细介绍日本案例
✅ 对比中国与日本
✅ 指出关键差异
✅ 说明中国优势
✅ 给出具体建议
✅ 列举每一代人的机会
```

### 🎯 适用场景

- ✅ 求职受挫，信心不足

- ✅ 想放弃或躺平

